Integrasi Google Spreadsheet

Masukkan URL Web App dari Google Apps Script yang terhubung ke Spreadsheet Anda.

Panduan Setup

1
Buat Google Spreadsheet

Buat spreadsheet baru dengan 2 sheet: "Stock" dan "Riwayat"

2
Setup Apps Script

Buka Extensions → Apps Script, lalu tempel kode yang tersedia

3
Deploy Web App

Deploy sebagai Web App dan salin URL-nya

4
Tempel URL

Paste URL ke kolom di sebelah kiri dan simpan
